Quick Start Guide

Follow these simple steps to get started with the Advanced R Data Visualization Suite:

  1. Install the Package:
    Use CRAN or GitHub to install the latest version.
    Make sure you have R version 4.0.0 or higher installed. View system requirements
  2. Load the Library:
    Load the package using library(advancedRviz).
    Pro tip: Add this to your .Rprofile for automatic loading.
  3. Launch the App:
    Start the interactive dashboard with launchApp().
    New Now supports remote deployment via Shiny Server!
  4. Import Your Data:
    Upload your datasets in various formats.
    • CSV files
    • Excel spreadsheets
    • R data files (RDS)
    • Feather files
  5. Explore Visualizations:
    Choose from our visualization gallery and customize as needed.
  6. Analyze & Export:
    Perform statistical analyses and export your findings.
    PDF HTML PNG SVG

For detailed instructions and advanced usage, refer to our Documentation.
